# 말 경매 프로젝트
말을 실시간 경매를 통해 판매,구입,관리할 수 있는 서비스 입니다.

## 서비스 기능
+ 회원기능 (회원가입,로그인,회원수정 및 탈퇴)
+ 말 등록 및 수정,삭제 기능 
+ 채팅 기능
+ 경매 진행 및 낙찰 기능
+ 나의 낙찰목록 csv 내려받기
